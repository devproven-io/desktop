# DevProven Desktop

Desktop analyzer for [DevProven](https://devproven.com) — turn your code into a verified developer skill profile.

## Download

Download the latest release from the [Releases](https://github.com/devproven-io/desktop/releases) page.

### Windows
- **DevProven_x64-setup.exe** — Recommended installer (NSIS)
- **DevProven_x64_en-US.msi** — MSI installer

### macOS
- Coming soon

## What's included

- DevProven desktop GUI
- Code analyzer (self-contained, no runtime needed)
- Browser-based login (Sign in with DevProven)
- API key fallback

## Getting started

1. Install DevProven Desktop
2. Click "Sign in with browser"
3. Approve in your browser
4. Add a repository and run analysis
5. View your Developer DNA at devproven.com

## Requirements

- Windows 10/11 x64 (macOS coming soon)
- Internet connection (for login and upload)
